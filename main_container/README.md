# Tiger Insights: Exploring Biology, Behavior, and Conservation

This project provides a comprehensive multi-page blog about tigers, covering their biology, behavior, habitat, conservation status, cultural significance, ecological role, and interesting facts.

## Features

- **Multi-page Structure**: Eight detailed pages covering different aspects of tigers
- **Navigation System**: Uses React Router for seamless navigation between pages
- **Responsive Design**: Works on all device sizes with optimized layouts
- **Rich Content**: Detailed information with well-structured formatting

## Pages Included

1. **Introduction**: Overview of tigers and their significance
2. **Biology and Subspecies**: Physical characteristics, adaptations, and tiger subspecies
3. **Behavior and Social Structure**: Social behavior, territorial habits, hunting, and family life
4. **Habitat and Range**: Historic and current distribution, habitat types, and requirements
5. **Conservation Status and Threats**: Current status, threats, and conservation efforts
6. **Tigers in Culture & Mythology**: Cultural significance across Asian religions and modern culture
7. **Role in the Ecosystem**: Ecological importance as apex predators
8. **Interesting Facts**: Collection of fascinating tiger facts

## Getting Started

In the project directory, you can run:

### `npm start`

Runs the app in development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### `npm test`

Launches the test runner in interactive watch mode.

### `npm run build`

Builds the app for production to the `build` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.
